# Loja_Artigos_Esportivos

Esse sistema web foi desenvolvido para simular um site de uma loja que vende produtos esportivos. O site possuí todos os componentes principais que compõem um site de alta performance. Foi utilizado HTML e CSS para desenvolver a página web.

Observação: O projeto está em desenvolviemnto. Ainda planejo fazer um sistema que irá deixar o site responsivo, além de adicionais outras páginas no site.
